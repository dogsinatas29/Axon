# AXON: 에이전트 오케스트레이션 프레임워크

AXON은 아키텍트, 시니어, 주니어로 구성된 AI 에이전트 군단이 명세서를 실제 고품질 코드로 변환하는 자율형 소프트웨어 생산 시스템입니다.

## 🚀 주요 원칙

1.  **아키텍처 우선**: 검증된 `architecture.md` (SSOT) 없이는 단 한 줄의 코드도 작성하지 않습니다.
2.  **역할의 성역**: 엄격한 계층 구조를 통해 코드의 품질과 책임 소재를 명확히 합니다.
3.  **상태 없는 실행**: 에이전트는 공장의 숙련된 부품처럼 교체 가능하며 독립적으로 작동합니다.

## 🏭 퀵 스타트: 지능형 공장 가동 (v0.0.14+)

AXON은 이제 대화형 부트스트래핑을 지원합니다. 더 이상 여러 개의 터미널 탭을 수동으로 관리할 필요가 없습니다.

1.  **준비**: API 키를 설정하세요 (예: `export GEMINI_API_KEY=your_key`).
2.  **가동**: 데몬 실행 커맨드를 입력하세요:
    ```bash
    cargo run -- run
    ```
3.  **인사 관리 (HR Selection)**: 역할별로 지능(Model)을 선택하세요:
    *   **Architect (CTO)**: `Architecture.md`라는 성역을 설계할 1명의 핵심 지능.
    *   **Seniors (Teams)**: 코드 검수 및 주니어를 관리할 인원과 지능.
    *   **Juniors (Workers)**: 실제 코드를 구현할 인원과 지능.
4.  **명세 투입**: 작업의 근거가 될 명세 파일 경로를 입력하세요 (예: `TEST/mile_stone/v0.0.1.md`).
5.  **공정 관제**: 아키텍트가 명세를 분석하여 자동으로 설계를 완료하고 작업을 배분하면, 시니어와 주니어들이 즉시 가동을 시작합니다.
