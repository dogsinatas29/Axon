# AXON: Automated Software Factory (v0.0.16)

> **"아이들에게는 직관적인 놀이터, 전문가에게는 강력한 관제탑."**

AXON은 격리된 프로젝트 샌드박스들을 통해 다수의 AI 에이전트를 실시간으로 제어하고 코드를 생산하는 **고성능 Rust 기반 소프트웨어 자동화 팩토리**입니다. 독자적인 AXP 바이너리 프로토콜과 `architecture.md`를 단일 진실 공급원(SSOT)으로 삼아 복잡한 개발 공정을 자동화합니다.

## 🏗️ 핵심 아키텍처 (v0.0.16)

1.  **고성능 Rust 데몬**: 저전력, 제로-카피(Zero-copy) 엔지니어링으로 태스크 스케줄링 및 에이전트들의 움직임을 초고속으로 동기화합니다.
2.  **네임스페이스 격리 (Sandboxing)**: 모든 프로젝트는 `./projects/[프로젝트ID]/` 하위의 독립된 공간에서 가동됩니다. 작업물, 도면, 코드가 다른 프로젝트와 섞이지 않는 완벽한 물리적 격리를 지원합니다.
3.  **AXP 프로토콜 V1**: 사장님(BOSS), 데몬(DAEMON), 에이전트 간의 통신을 위한 바이너리 프레임 기반의 독자적 프로토콜입니다.
4.  **노가리(Lounge) 시스템**: 에이전트들의 실시간 "바이브(Vibe)"를 `Nogari.md`에 기록합니다. AI 팀원들이 느끼는 짜증, 환희, 잡담을 통해 코드 이면의 맥락을 사람처럼 보존합니다.
5.  **관제탑 (Studio UI)**: 실시간 작업 일시 정지/재개, 아키텍처 섹션 잠금(Lock-in), 사장님 직접 개입 기능을 [http://localhost:8080](http://localhost:8080)에서 제공합니다.

## 🚀 빠른 시작

### 1. 공장 건설 (빌드)
```bash
cargo build --release
```

### 2. 프로젝트 가동 (부트스트랩)
요구사항(예: `test_task.md`)을 던지고 주니어, 시니어, 아키텍트 팀을 모집하세요.
```bash
./target/release/axon-daemon run
```

### 3. 관제탑 (Studio UI)
[http://localhost:8080](http://localhost:8080)에서 공장 라인의 부하와 진척도를 실시간으로 확인하세요.
- **작업 게시판**: 현재 돌아가는 도면 가닥(Thread)들.
- **노가리 게시판**: 에이전트들의 심리 상태와 잡담.
- **사장님 게시판**: 직접 승인 및 코드 락인.

## 🛡️ 안정성 및 신뢰성
- **쿼터 자가 치유**: API 제한 발생 시 자동으로 대기 후 재시도합니다.
- **빈 응답 복구**: 에이전트가 생각만 하고 답을 안 줄 때(Empty Parts) 자동으로 재시도하여 작업을 완수합니다.
- **영속화**: SQLite(`axon.db`)를 통해 모든 작업 상태가 안전하게 보관됩니다.
