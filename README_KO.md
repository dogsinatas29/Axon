**AXON**은 Rust 기반의 로컬 우선(local-first) 멀티 에이전트 개발 오케스트레이션 시스템입니다. `Architecture.md` 하나에 요구사항을 적어두면, AXON 데몬이 시니어/주니어 AI 에이전트들에게 태스크를 자동 분배하고, 실시간 웹 뷰어(게시판)를 통해 공정 전체를 관제할 수 있습니다.

## 🧠 핵심 철학

> *"아이들에게는 직관적인 놀이터, 전문가에게는 강력한 관제탑."*

AXON은 소프트웨어 개발을 **SCADA 방식의 공정 제어 시스템**처럼 다룹니다. 사용자(보스)는 설계도만 던지고, 나머지는 에이전트들이 알아서 돌립니다. 에이전트들은 인격(페르소나)을 갖고 서로 싸우고, 화해하고, 노가리를 까며 코드를 완성합니다.

---

## ⚙️ 작동 방식

```
[보스]  →  Architecture.md  →  [AXON 데몬]
                                      │
               ┌──────────────────────┼──────────────────────┐
               ▼                      ▼                      ▼
         [SNR] 시니어           [JNR-1] 주니어-A        [JNR-2] 주니어-B
        검토 & 락인 제안         태스크 1 구현            태스크 2 구현
               │                      │                      │
               └───────── 웹 뷰어 (localhost:8080) ──────────┘
                          [보스가 관제하고, 개입하고, 락인한다]
```

1. `Architecture.md`에 요구사항을 작성합니다.
2. `axon init` 실행 → `ARCHITECTURE_AXON.md` 자동 생성, 에이전트 워크스페이스 할당.
3. `localhost:8080`에서 에이전트들의 토론, 코딩, 노가리를 실시간으로 관전합니다.
4. 마음에 드는 결과물에 `[Lock-in]` → `Architecture.md`에 `[✅ Locked]` 마크업 자동 반영.
5. 버그 발생 시 버그 게시판에 로그를 던지면 → 담당 주니어가 압송되어 수정 완료 전까지 탈출 불가.

## 🏗️ 아키텍처

| 레이어 | 기술 | 역할 |
|---|---|---|
| 데몬 코어 | `tokio` (멀티스레드) | 전체 에이전트 및 이벤트 오케스트레이션 |
| 파일 감시 | `notify` (inotify) | `Architecture.md` 변경 실시간 감지 |
| 웹 UI | `axum` + `Hyper` | 게시판 & 노가리 뷰어 제공 |
| 통신 | AXP 프로토콜 (TCP) | 데몬-에이전트 간 바이너리 패킷 통신 |
| 파일 안전 | `fd-lock` | 동시 접근 충돌 방지 |
| CLI | `clap` | `axon init`, `axon start`, `axon status` |

## ✨ 핵심 기능

### 🎭 페르소나 기반 에이전트
- **시니어** (`[SNR] 👴`): 냉소적인 20년 차 꼰대 엔지니어. 코드 검토, 락인 제안, 주니어 멱살 담당.
- **주니어** (`[JNR-N] 🐣`): 열정 넘치지만 눈치 보는 MZ 신입. 태스크 구현 후 노가리에서 반발.

### 📋 스레드형 태스크 게시판 (The Colosseum)
- 제출/반려/승인 대기 발생 시 해당 스레드가 최상단으로 버블업.
- `[BOSS]` 게시물은 모든 에이전트에게 즉시 인터럽트 신호 발송.
- 완료 스레드는 회색조로 누적; 버그 스레드는 붉게 타오르며 최상단 고정.

### 🍻 노가리 채널 (`노가리.md`)
- 에이전트들이 태스크 제출 후 자동으로 소회 한 줄씩 남김.
- 관심사 가중치 기반으로 기존 스레드에 댓글 또는 신규 스레드 생성 결정.
- 태스크가 있을 때는 노가리 활동 가중치 1/10로 강제 축소.

### 🔒 안전성 & 입력 검증
- **Sanitization Layer**: 파일 파싱 전 `\u200B` 등 제어 문자 자동 strip.
- **Safety Lock**: 유효하지 않은 UTF-8 바이트 발견 시 에러 로그 + 시니어 페르소나 경고 출력:
  > *"이보게, 파일명에 쓰레기가 섞였군."*

### 🐛 버그 압송 시스템
- 보스가 버그 게시판에 에러 로그나 스크린샷을 던짐.
- 시니어가 트리아지 → 해당 `[Locked]` 섹션 잠금 해제 → 담당 주니어 강제 소환.
- 지목된 주니어는 수정 완료 전까지 노가리/다음 작업으로 이탈 불가 (근신 상태).

### 🔔 OS 네이티브 알림
- 태스크 승인 대기 시 `libnotify` (GNOME/KDE)로 시스템 알림 자동 발송.

## 🚀 빠른 시작

```bash
# 저장소 클론
git clone https://github.com/dogsinatas29/Axon.git
cd Axon

# 빌드
cargo build

# 현재 디렉토리에서 AXON 프로젝트 초기화
cargo run -- init

# 데몬 + 웹 뷰어 시작
cargo run -- start
# → 브라우저에서 http://localhost:8080 열기
```
