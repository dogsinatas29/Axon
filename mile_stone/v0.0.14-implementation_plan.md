# 🚀 Implementation Plan: COT/TOT Reasoning Enforcement (v0.0.14)

## 🏛️ Objective
"AI 에이전트의 단순 응답 모델을 탈피하고, 명시적인 사고 단계(Thought Process)를 통해 고도화된 아키텍처와 코드를 생산한다."

---

## 🛠️ 작업 단계 (Action Items)

### 1. 부트스트랩 아키텍트 지능 강화 (Daemon Layer)
- **대상**: `crates/axon-daemon/src/lib.rs` -> `bootstrap_from_spec` (216-222행)
- **실제 증상**: "Generate architecture..." 식의 단순 명령만 존재.
- **변경 사항 (사고의 해방)**:
    - 프롬프트 최상단에 **Sovereign Protocol (v0.2.21+)** 및 `Hub -> Cluster -> Node` 구조를 철저히 박음.
    - `STEP 1: ANALYSIS (COT)` 및 `STEP 2: EVALUATION (TOT)` 절차를 명시하여 아키텍트의 '날림' 설계를 방지.

### 2. 에이전트 엔진 사고 체계 수립 (Agent Layer)
- **대상**: `crates/axon-agent/src/lib.rs`
- **실제 증상**: Junior(58행), Senior(146행) 등에 `DO NOT EXPLAIN` 등의 억압적 지시 존재.
- **변경 사항 (지능 복구)**:
    - **역할별 사고 체계 의무화**:
        - **Junior (`process_task`)**: `<thought>` (COT) 태그 강제.
        - **Senior (`review_proposal`)**: `<analysis>` (TOT) 태그 강제.
        - **Architect (`validate_architecture`)**: `<reasoning>` (COT+TOT) 태그 강제.
    - **독소 조항 삭제**: 에이전트의 사고를 막는 모든 '입막음' 지시를 제거.

### 3. 결과 파서(Result Parser) 고도화
- **내용**: 에이전트가 뱉어내는 `<thought>` 혹은 `<analysis>` 태그를 감지하여, 실제 결과물(코드/JSON)만 깔끔하게 추출하는 로직 정교화.

---

## 🧪 검증 시나리오 (Validation Scenario)
1.  **Bootstrapping**: `GEMINI.md`를 넣었을 때, `architecture.md`가 Hub/Cluster/Node 구조로 자동 생성되는지 확인.
2.  **Junior Task**: 주니어가 코드를 짜기 전 "왜 이 라이브러리를 썼는지"에 대한 사고 과정이 로그에 출력되는지 확인.
3.  **Senior Review**: "성능 면에서는 합격이나, 보안상 이런 경로가 우려된다"는 식의 다각도 검토가 이루어지는지 확인.

---

## ⚠️ 주의 사항
- 토큰 소모 증가: 사고 단계가 추가됨에 따라 더 많은 컨텍스트 토큰이 소모됨을 인지하고 모니터링할 것.
- JSON 파싱 오류: 설명이 추가되면서 JSON 블록을 찾는 정규식이나 파서가 흔들리지 않도록 방어할 것.

---

## 🚨 추가 후속 조치 (Next Steps: UI Data Binding & DB Connection)
- **통합 일감 워크 보드 연결**: `/api/tasks` 하드코딩 빈 배열(`vec![]`)을 삭제하고, `daemon.storage.list_all_tasks()` 등 실제 DB 조회를 통해 일감을 가져오도록 백엔드 구현.
- **조직도(Office) 인사 기록 영속화**: 터미널에서 고용(hire)한 에이전트들을 `daemon.storage.save_agent()`를 통해 SQLite에 실제로 Create(생성) 반영.
- **예외(Error) 시그널링**: 에이전트가 처리 도중 API 요금/쿼터 한도 초과(`Quota Exceeded`) 같은 중대한 에러 직면 시, 침묵하지 않고 웹소켓에 '에러 발생' 시그널(`SystemWarning`/`Error`)을 Publish 하여 화면에 즉시 띄우기.
