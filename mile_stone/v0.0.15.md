## 🧠 상세 설계 및 로직

### 1. 🚦 전역 속도 조율기 (Global Rate Limiter / Throttle)
- **설계 의도**: 다수의 주니어 에이전트가 동시에 깨어나 구글 API에 무결재 돌격하는 것을 방지.
- **구현 방식**: 
  - `axon_daemon` 내부의 작업 할당 스케줄러에 시간 기반 (4.1초당 1건) 제동 장치 추가. [✅ Completed]

### 2. 🛡️ 429 Quota Error 핸들러 및 자동 재개 (Retry Backoff)
- **설계 의도**: 만약 쿼터 초과를 맞더라도 에이전트가 패닉에 빠지지 않고 유연하게 대처.
- **구현 방식**: 
  - `axon_model`에서 `QUOTA_WAIT:N` 에러를 반환하고, `AgentRuntime`이 이를 가로채 `SystemWarning` 이벤트 발행 후 대기 및 재시도. [✅ Completed]

### 3. 🚨 시각적 에러 시그널링 (Web UI WebSocket)
- **설계 의도**: 파업 중인(?) 주니어들의 상태를 관제탑 메뉴얼과 스레드 창에 명확히 표출.
- **구현 방식**:
  - `EventBus`를 통해 `SystemWarning` 이벤트를 UI 실시간 시그널 탭으로 전송. [✅ Completed]

### 4. ♻️ 태스크 자동 복구 (Resume Task Recovery)
- **설계 의도**: `--resume` 시 DB에 저장된 완료되지 않은 태스크들이 멈춰있지 않고 즉시 재개되도록 함.
- **구현 방식**:
  - `Daemon::run` 시작 시 DB에서 `Pending/InProgress` 상태의 태스크를 전역 큐에 재진입시킴. [✅ Completed]

### 5. 👁️ Studio UI 코드 가독성 개선 (Patch Rendering & Unescape)
- **설계 의도**: 주니어가 제출한 코드의 줄바꿈(`\n`), 탭(`\t`)이 그대로 글자로 노출되는 현상을 해결하고 Diff 형식을 예쁘게 렌더링.
- **구현 방식**:
  - `ThreadDetail.tsx`의 텍스트 렌더링 로직에 `Unescape` 처리(줄바꿈 실제 반영) 적용.
  - `@@` 시작 텍스트를 감지하여 전용 코드 블록 하이라이트 UI로 감싸기. [🏗️ Planned]
