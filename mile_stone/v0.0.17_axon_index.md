## ⚠️ SYSTEM DIRECTIVE

You are NOT allowed to:

- Skip any phase
- Change execution order
- Assume completion
- Infer missing steps

You MUST:

- Load phases strictly in order
- Complete each phase before proceeding
- Pass assertion before unlocking next phase

## 🔒 PHASE LOCK RULE

- Each phase is LOCKED by default
- Next phase is UNLOCKED only if:
  - Output is complete
  - Assertion is satisfied

---

## 🧩 LOADING RULE

- Only one phase can be active at a time
- Next phase MUST NOT be loaded early
- Previous phase MUST NOT be revisited unless failure occurs

## 🧠 SUMMARY

AXON은 자동화 시스템이 아니다.

- 사용자 정의 기반 실행
- 단계별 검증 기반 진행
- 명시적 상태 전이 기반 시스템

이 문서는 LLM을:

"텍스트 해석자" → "단계적 실행 엔진"
