# AXON v0.0.18 3-Way Merge Design

## 🎯 PURPOSE

Introduce a state-preserving merge system using BASE, CURRENT, and NEW
states.

------------------------------------------------------------------------

## 1. MODEL

BASE = last successful commit\
CURRENT = filesystem state\
NEW = LLM output

MERGED = merge(BASE, CURRENT, NEW)

------------------------------------------------------------------------

## 2. FILE CASES

### Case 1: New File

BASE: 없음\
CURRENT: 없음\
NEW: 있음\
→ Add

### Case 2: Unchanged

BASE == CURRENT == NEW\
→ Skip

### Case 3: Safe Update

BASE == CURRENT != NEW\
→ Overwrite

### Case 4: Conflict

BASE != CURRENT\
BASE != NEW\
CURRENT != NEW\
→ Conflict

------------------------------------------------------------------------

## 3. MERGE LOGIC

``` python
def merge_file(base, current, new):
    if current == base:
        return new

    if new == base:
        return current

    if current == new:
        return current

    return "CONFLICT"
```

------------------------------------------------------------------------

## 4. PIPELINE

LLM OUTPUT (NEW) → Parser → Code Validation → Dependency Validation →
3-Way Merge → Conflict → Retry → Atomic Commit → Update BASE → CLOSED

------------------------------------------------------------------------

## 5. CONFLICT STRATEGY

### Initial (MVP)

-   Fail-fast
-   Trigger retry

### Advanced

-   LLM-assisted merge
-   Inline conflict markers (optional)

------------------------------------------------------------------------

## 6. RETRY TEMPLATE

Reason: MERGE CONFLICT

Provide: - BASE version - CURRENT version - NEW version

Goal: Produce a single resolved file.

Follow OUTPUT CONTRACT strictly.

------------------------------------------------------------------------

## 7. CORE PRINCIPLE

-   Preserve valid state
-   Prevent overwrite loss
-   Detect and resolve conflicts explicitly

------------------------------------------------------------------------

## RESULT

-   Safe incremental evolution
-   No silent overwrite
-   Deterministic state transitions
