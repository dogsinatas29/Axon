# AXON v0.0.18 Class / Method Extension

## PURPOSE

Extend function-level system to class + method granularity

## TARGETS

-   ClassDef
-   FunctionDef inside class

## STRATEGY

-   Extract class span
-   Inside class, extract method spans
-   Apply same replacement logic

## EDGE CASES

-   Nested classes
-   Decorated methods
-   Static / class methods

## RESULT

Fine-grained control at full object structure level
