# 1️⃣ MULTI-FILE DEPENDENCY VALIDATION

## Goal

Guarantee project-level integrity (not just per-file correctness)

------------------------------------------------------------------------

## 1.1 Import Extraction

``` python
import ast

def extract_imports(code):
    tree = ast.parse(code)
    imports = []

    for node in ast.walk(tree):
        if isinstance(node, ast.Import):
            for n in node.names:
                imports.append(n.name.split('.')[0])

        elif isinstance(node, ast.ImportFrom):
            if node.module:
                imports.append(node.module.split('.')[0])

    return imports
```

------------------------------------------------------------------------

## 1.2 Dependency Validation

``` python
def validate_dependencies(files):
    filenames = {name.replace(".py", "") for name, _ in files}

    for name, code in files:
        imports = extract_imports(code)

        for imp in imports:
            if imp in filenames:
                continue  # local dependency OK
            # external imports allowed (configurable)

    return True
```

------------------------------------------------------------------------

## 1.3 Entry Point Check

``` python
def has_entry_point(files):
    for name, code in files:
        if "if __name__ == '__main__'" in code:
            return True
    return False
```

------------------------------------------------------------------------

## 1.4 Cycle Detection (Optional Advanced)

``` python
def build_graph(files):
    graph = {}
    filenames = {name.replace(".py", "") for name, _ in files}

    for name, code in files:
        key = name.replace(".py", "")
        graph[key] = []

        for imp in extract_imports(code):
            if imp in filenames:
                graph[key].append(imp)

    return graph
```

# 2️⃣ DIFF-BASED INCREMENTAL COMMIT

## Goal

Apply minimal changes while preserving valid project state

------------------------------------------------------------------------

## 2.1 Strategy

-   Compare current state vs new state
-   Apply only changed files
-   Preserve unchanged files

------------------------------------------------------------------------

## 2.2 Diff Detection

``` python
def compute_diff(old_files, new_files):
    old_map = dict(old_files)
    new_map = dict(new_files)

    added = []
    modified = []
    deleted = []

    for name in new_map:
        if name not in old_map:
            added.append(name)
        elif old_map[name] != new_map[name]:
            modified.append(name)

    for name in old_map:
        if name not in new_map:
            deleted.append(name)

    return added, modified, deleted
```

------------------------------------------------------------------------

## 2.3 Incremental Apply

``` python
import os

def apply_diff(base_dir, old_files, new_files):
    added, modified, deleted = compute_diff(old_files, new_files)

    for name in deleted:
        path = os.path.join(base_dir, name)
        if os.path.exists(path):
            os.remove(path)

    for name in added + modified:
        code = dict(new_files)[name]
        path = os.path.join(base_dir, name)

        os.makedirs(os.path.dirname(path), exist_ok=True)

        with open(path, "w", encoding="utf-8") as f:
            f.write(code)
```

------------------------------------------------------------------------

## 2.4 Safety Rule

-   Diff 적용 전 반드시 dependency validation 통과
-   실패 시 diff 적용 금지

# 3️⃣ PIPELINE INTEGRATION

``` python
def process(files, current_state, target_dir):
    if not validate_dependencies(files):
        return "FAIL_DEPENDENCY"

    if not has_entry_point(files):
        return "FAIL_ENTRY"

    apply_diff(target_dir, current_state, files)

    return "SUCCESS"
```
