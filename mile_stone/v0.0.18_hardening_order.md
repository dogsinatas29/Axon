# AXON v0.0.18 Hardening Order

## Implementation Priority

### 1. Atomicity (First)

Ensures all-or-nothing file generation.

``` python
def atomic_commit(files, target_dir):
    tmp_dir = create_temp_dir()
    try:
        for name, code in files:
            write_file(tmp_dir, name, code)

        validate_files(tmp_dir)
        move_all(tmp_dir, target_dir)
        return True
    except Exception:
        rollback(tmp_dir)
        return False
```

------------------------------------------------------------------------

### 2. Code Validity (Second)

Prevents invalid or non-executable code.

``` python
compile(code, "<string>", "exec")
```
