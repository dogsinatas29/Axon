# 2. AST-BASED IMPORT ANALYSIS (CORE UPGRADE)

## Problem with Regex

-   Fails on alias imports
-   Breaks on multiline imports
-   Cannot handle relative imports

------------------------------------------------------------------------

## 2.1 AST Import Extraction

``` python
import ast

def extract_imports_ast(code):
    tree = ast.parse(code)
    imports = []

    for node in ast.walk(tree):

        if isinstance(node, ast.Import):
            for alias in node.names:
                imports.append({
                    "type": "import",
                    "module": alias.name,
                    "alias": alias.asname
                })

        elif isinstance(node, ast.ImportFrom):
            imports.append({
                "type": "from",
                "module": node.module,
                "names": [n.name for n in node.names],
                "level": node.level
            })

    return imports
```

# 3. DEPENDENCY ANALYSIS

## 3.1 Module → File Mapping

``` python
def module_to_file(module):
    if module is None:
        return None
    return module.split(".")[-1] + ".py"
```

------------------------------------------------------------------------

## 3.2 Missing Module Detection

``` python
def find_missing_modules_ast(files):
    filenames = {name.replace(".py", "") for name, _ in files}
    missing = set()

    for name, code in files:
        imports = extract_imports_ast(code)

        for imp in imports:
            if imp["type"] == "import":
                mod = imp["module"].split(".")[0]
            else:
                mod = imp["module"]

            if mod and mod not in filenames:
                missing.add(mod)

    return missing
```

# 4. AST-BASED FILENAME INFERENCE

``` python
def match_module_to_code(module, code):
    if f"class {module.capitalize()}" in code:
        return True
    if f"def {module}" in code:
        return True
    return False
```

------------------------------------------------------------------------

## 4.1 Filename Correction

``` python
def fix_filenames_with_ast(files, missing):
    updated = []

    for name, code in files:
        for mod in missing:
            if match_module_to_code(mod, code):
                name = f"{mod}.py"
        updated.append((name, code))

    return updated
```
