## 2.2 Inference Logic

``` python
def infer_filename(code, context):
    if context.get("expected_files"):
        return context["expected_files"][0]

    if "class Database" in code:
        return "database.py"

    if "validator" in code.lower():
        return "validators.py"

    return f"{context['task_id']}.py"
```

------------------------------------------------------------------------

## 2.3 Collision Safety

``` python
def ensure_unique(name, existing):
    if name not in existing:
        return name

    i = 1
    while f"{name}_{i}" in existing:
        i += 1

    return f"{name}_{i}"
```

------------------------------------------------------------------------

## 2.4 Final Heuristic Parser

``` python
def parse_heuristic(output, context, existing_files):
    blocks = extract_code_blocks(output)
    largest = max(blocks, key=len)

    filename = infer_filename(largest, context)
    filename = ensure_unique(filename, existing_files)

    return [(filename, largest)]
```
