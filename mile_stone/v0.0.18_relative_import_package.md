# 5. RELATIVE IMPORT RESOLUTION

## Rule

``` text
from .x       → same directory
from ..x      → parent directory
from ...x     → ancestor levels
```

------------------------------------------------------------------------

## Resolver

``` python
def resolve_relative(module, level, current_path):
    parts = current_path.split("/")[:-1]

    base = parts[:len(parts) - level + 1]

    if module:
        base += module.split(".")

    return "/".join(base) + ".py"
```

------------------------------------------------------------------------

# 6. ABSOLUTE IMPORT HANDLING

``` python
def resolve_absolute(module):
    return module.replace(".", "/") + ".py"
```

------------------------------------------------------------------------

# 7. INTEGRATED RESOLUTION

``` python
def resolve_import(imp, current_path):
    if imp["type"] == "import":
        return resolve_absolute(imp["module"])

    if imp["level"] > 0:
        return resolve_relative(imp["module"], imp["level"], current_path)

    return resolve_absolute(imp["module"])
```

------------------------------------------------------------------------

# 8. DIRECTORY RECONSTRUCTION

## Goal

Convert flat file list → hierarchical structure

------------------------------------------------------------------------

## Strategy

``` python
def build_tree(files):
    tree = {}

    for path, code in files:
        parts = path.split("/")
        node = tree

        for p in parts[:-1]:
            node = node.setdefault(p, {})

        node[parts[-1]] = code

    return tree
```

------------------------------------------------------------------------

# 9. DEPENDENCY VALIDATION (PATH-AWARE)

``` python
def validate_paths(files):
    paths = {path for path, _ in files}

    for path, code in files:
        imports = extract_imports_ast(code)

        for imp in imports:
            target = resolve_import(imp, path)

            if target not in paths:
                return False

    return True
```
