# 1️⃣ SEMANTIC MERGE (AST 기반)

## Goal

Preserve meaning, not just text

------------------------------------------------------------------------

## 1.1 Core Idea

-   Parse code into AST
-   Compare structure instead of raw text
-   Merge at function/class level

------------------------------------------------------------------------

## 1.2 AST Parsing

``` python
import ast

def parse_ast(code):
    return ast.parse(code)
```

------------------------------------------------------------------------

## 1.3 Function Extraction

``` python
def extract_functions(code):
    tree = ast.parse(code)
    functions = {}

    for node in tree.body:
        if isinstance(node, ast.FunctionDef):
            functions[node.name] = node

    return functions
```

------------------------------------------------------------------------

## 1.4 AST → Code

``` python
import astor  # or ast.unparse (Python 3.9+)

def to_code(node):
    return ast.unparse(node)
```

# 2️⃣ FUNCTION-LEVEL DIFF

## Goal

Detect changes per function instead of file

------------------------------------------------------------------------

## 2.1 Diff Strategy

``` python
def diff_functions(base_funcs, new_funcs):
    added = []
    modified = []
    unchanged = []

    for name in new_funcs:
        if name not in base_funcs:
            added.append(name)
        elif ast.dump(base_funcs[name]) != ast.dump(new_funcs[name]):
            modified.append(name)
        else:
            unchanged.append(name)

    return added, modified, unchanged
```

------------------------------------------------------------------------

# 3️⃣ SEMANTIC MERGE LOGIC

## 3.1 Merge Function Unit

``` python
def merge_function(base, current, new):
    if current == base:
        return new

    if new == base:
        return current

    if current == new:
        return current

    return None  # conflict
```

------------------------------------------------------------------------

## 3.2 Merge All Functions

``` python
def merge_functions(base_code, current_code, new_code):
    base_funcs = extract_functions(base_code)
    current_funcs = extract_functions(current_code)
    new_funcs = extract_functions(new_code)

    merged = {}

    all_names = set(base_funcs) | set(current_funcs) | set(new_funcs)

    for name in all_names:
        b = base_funcs.get(name)
        c = current_funcs.get(name)
        n = new_funcs.get(name)

        result = merge_function(
            ast.dump(b) if b else None,
            ast.dump(c) if c else None,
            ast.dump(n) if n else None
        )

        if result is None:
            return None

        merged[name] = n or c

    return merged
```

# 4️⃣ RECONSTRUCTION

## Goal

Rebuild full file from merged functions

``` python
def rebuild_file(func_map):
    code = ""

    for name, node in func_map.items():
        code += ast.unparse(node) + "\n\n"

    return code
```
