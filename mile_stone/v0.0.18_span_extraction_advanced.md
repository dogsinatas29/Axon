## 🎯 PURPOSE

Accurately extract function/class code spans while preserving structure,
decorators, and comments.

------------------------------------------------------------------------

# 1. CORE IDEA

Use: - AST → semantic structure (what) - tokenize → exact source mapping
(where)

# 2. BASE ALGORITHM

## Step 1: Parse AST

``` python

tree = ast.parse(source)
```

------------------------------------------------------------------------

## Step 2: Locate target node

``` python
for node in ast.walk(tree):
```

------------------------------------------------------------------------

## Step 3: Initial span

------------------------------------------------------------------------

# 3. DECORATOR HANDLING

## Problem

Decorators exist ABOVE lineno

## Fix

``` python
if node.decorator_list:
    start = min(d.lineno for d in node.decorator_list)
```

------------------------------------------------------------------------

# 4. TOKEN-LEVEL REFINEMENT

## Goal

Convert line-based span → exact character span

------------------------------------------------------------------------

## Token scan

``` python
import tokenize
from io import StringIO

tokens = list(tokenize.generate_tokens(StringIO(source).readline))
```

------------------------------------------------------------------------

## Map line → char index

``` python
lines = source.splitlines(keepends=True)

def line_to_offset(line_no):
    return sum(len(lines[i]) for i in range(line_no - 1))
```

------------------------------------------------------------------------

## Final span

``` python
start_offset = line_to_offset(start)
end_offset = line_to_offset(end + 1)
```

------------------------------------------------------------------------

# 5. EXTRA SAFETY (INDENT CHECK)

## Problem

end_lineno may include trailing blocks incorrectly

## Fix

Check indentation:

``` python
def get_indent(line):
    return len(line) - len(line.lstrip())
```

Ensure block ends when indent decreases.

# 7. FINAL EXTRACTION FUNCTION

``` python
def extract_span(source, node):
    lines = source.splitlines(keepends=True)

    start = node.lineno
    end = node.end_lineno

    if node.decorator_list:
        start = min(d.lineno for d in node.decorator_list)

    def offset(line):
        return sum(len(lines[i]) for i in range(line - 1))

    return offset(start), offset(end + 1)
```

------------------------------------------------------------------------

# 8. REPLACEMENT

``` python
def replace_span(source, start, end, new_code):
    return source[:start] + new_code + source[end:]
```

-   AST decides WHAT to replace
-   tokenize decides WHERE to replace
-   source slicing preserves HOW it looks
