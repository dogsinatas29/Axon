# AXON v0.0.18 Span-Based Merge

## PURPOSE

Apply precise modifications using extracted spans.

## FLOW

1.  Extract span (AST + tokenize)
2.  Replace only target region
3.  Preserve rest of file

## CORE LOGIC

def replace_span(src, start, end, new): return src\[:start\] + new +
src\[end:\]

## GUARANTEES

-   No structural corruption
-   Minimal diff
-   Safe incremental evolution
