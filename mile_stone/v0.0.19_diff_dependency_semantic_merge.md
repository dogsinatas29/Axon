# AXON v0.0.19 DIFF + DEPENDENCY VALIDATION + SEMANTIC MERGE

## PURPOSE

Enhance pipeline with: - Incremental diff commit - Multi-file dependency
validation - Semantic merge (AST-based)

------------------------------------------------------------------------

# 1. DIFF INCREMENTAL COMMIT

## FLOW

OLD FILE → NEW OUTPUT → DIFF GENERATION → VALIDATION → APPLY PATCH

## RULES

-   No full overwrite
-   Only changed sections updated
-   Must pass syntax validation

------------------------------------------------------------------------

# 2. MULTI-FILE DEPENDENCY VALIDATION

## CHECKS

-   Import existence
-   Circular dependency detection
-   Missing modules
-   Function existence

------------------------------------------------------------------------

# 3. SEMANTIC MERGE (AST BASED)

## APPROACH

-   Parse code into AST
-   Compare function/class units
-   Merge only changed nodes

------------------------------------------------------------------------

## CONFLICT HANDLING

-   Detect overlapping edits
-   Send to LLM resolution loop
-   Re-validate after merge

------------------------------------------------------------------------

# 4. PIPELINE INTEGRATION

Junior Output → Parse → Dependency Validation → Diff Generation →
Semantic Merge → Final Validation → Commit

------------------------------------------------------------------------

# 5. SUCCESS CONDITIONS

-   No broken imports
-   No syntax errors
-   Minimal diff applied
-   Merge conflicts resolved

------------------------------------------------------------------------

# RESULT

Safe incremental updates with structural awareness.
