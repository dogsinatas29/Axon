# AXON v0.0.19 EXECUTION HARNESS (RUN + TEST + ROLLBACK)

## PURPOSE

Provide a deterministic execution layer that: - runs generated code -
validates runtime correctness - rolls back on failure

------------------------------------------------------------------------

# 1. CORE FLOW

GENERATED CODE → Build → Execute → Test → Validate → Commit OR Rollback

------------------------------------------------------------------------

# 2. EXECUTION ENGINE

``` python
import subprocess
import tempfile
import os

def run_code(entry_file: str, timeout=5):
    try:
        result = subprocess.run(
            ["python", entry_file],
            capture_output=True,
            text=True,
            timeout=timeout
        )
        return {
            "success": result.returncode == 0,
            "stdout": result.stdout,
            "stderr": result.stderr
        }
    except Exception as e:
        return {"success": False, "error": str(e)}
```

------------------------------------------------------------------------

# 3. TEST HARNESS

## Minimal validation

``` python
def basic_test(output: dict):
    if not output["success"]:
        return False
    return True
```

## Optional: expected output

``` python
def assert_contains(output: str, keyword: str):
    return keyword in output
```

------------------------------------------------------------------------

# 4. SANDBOX EXECUTION

``` python
def sandbox_run(file_map: dict):
    with tempfile.TemporaryDirectory() as tmp:
        for fname, code in file_map.items():
            path = os.path.join(tmp, fname)
            with open(path, "w") as f:
                f.write(code)

        entry = os.path.join(tmp, "main.py")
        return run_code(entry)
```

------------------------------------------------------------------------

# 5. ROLLBACK SYSTEM

## Snapshot before commit

``` python
import shutil

def snapshot(files: list, backup_dir: str):
    os.makedirs(backup_dir, exist_ok=True)
    for f in files:
        if os.path.exists(f):
            shutil.copy2(f, os.path.join(backup_dir, f))
```

## Restore on failure

``` python
def rollback(backup_dir: str):
    for f in os.listdir(backup_dir):
        shutil.copy2(os.path.join(backup_dir, f), f)
```

------------------------------------------------------------------------

# 6. FULL HARNESS PIPELINE

``` python
def execution_harness(file_map: dict, existing_files: list):
    backup_dir = ".backup"

    # 1. snapshot
    snapshot(existing_files, backup_dir)

    # 2. sandbox run
    result = sandbox_run(file_map)

    # 3. test
    if not basic_test(result):
        rollback(backup_dir)
        raise RuntimeError("Execution failed, rollback complete")

    # 4. commit
    for fname, code in file_map.items():
        with open(fname, "w") as f:
            f.write(code)

    return result
```
