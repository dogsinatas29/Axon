# AXON v0.0.19 FULL CONTRACT CHAIN (PRODUCTION)

## PURPOSE

End-to-end deterministic LLM pipeline with strict validation and atomic
commit.

------------------------------------------------------------------------

## PIPELINE

Architect → Validator → Planner → Validator → Junior → Validator →
Commit

RULES: - Any failure → retry - Heuristic output → reject - Only
validated output → commit

------------------------------------------------------------------------

## CORE PRINCIPLES

1.  Do not trust LLM output
2.  Trust only format + validation
3.  Fail fast
4.  No heuristic commit
5.  Deterministic pipeline

------------------------------------------------------------------------

## COMPONENTS

### Architect

-   Input: GEMINI.md
-   Output: architecture.md

### Planner

-   Input: architecture.md
-   Output: task list

### Junior

-   Input: task
-   Output: code (strict contract)

------------------------------------------------------------------------

## VALIDATION

Each stage must pass strict validation or retry.

------------------------------------------------------------------------

## COMMIT

Atomic file write only after validation passes.

------------------------------------------------------------------------

## RESULT

Deterministic, safe, production-ready pipeline.
