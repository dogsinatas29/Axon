# 2. 🔁 PRODUCTION RETRY PROMPT

    ONLY OUTPUT EXACTLY IN THIS FORMAT:

    [OUTPUT]

    FILE: <filename>

    ```python
    <code>

RULES: - DO NOT include explanation - DO NOT include markdown text - DO
NOT include comments outside code - ONLY FILE + CODE

FAILURE TO FOLLOW FORMAT WILL RESULT IN RETRY


    ---

    # 3. 🧱 PARSER HARD FAIL CONDITIONS

    ```python
    def hard_fail(output: str):
        if "[OUTPUT]" not in output:
            return True

        if "FILE:" not in output:
            return True

        if "```" not in output:
            return True

        # detect explanation leakage
        forbidden = ["This code", "Explanation", "Below is", "Here is"]
        if any(word in output for word in forbidden):
            return True

        return False

Usage:

``` python
if hard_fail(output):
    retry()
```

------------------------------------------------------------------------

# 4. 🧪 STRICT FORMAT VALIDATION

``` python
def valid_format(output: str):
    return (
        "[OUTPUT]" in output and
        "FILE:" in output and
        "```" in output
    )
```
