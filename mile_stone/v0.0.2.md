# AXON Milestone v0.0.2 - Modularization & Persistence

## 🎯 핵심 목표
- 프로젝트를 모듈형 Rust Workspace로 재구축.
- SQLite 기반의 영속성 계층(Storage) 구현.
- 게시판(Board-driven) 워크플로우를 위한 스케줄러 개발.
- React 기반 AXON Studio 고도화.

## 🏗️ 아키텍처 사양
### 1. 프로젝트 구조 (crates/)
- **axon-core**: 공통 데이터 모델 및 이벤트 버스.
- **axon-daemon**: 메인 오케스트레이터 및 CLI.
- **axon-dispatcher**: 작업 분배 및 워커 스케줄링.
- **axon-agent**: 에이전트 실행 런타임 및 페르소나 엔진.
- **axon-model**: LLM 드라이버 추상화 레이어.
- **axon-storage**: SQLite 저장소 핸들러.

### 2. 주요 기능 로직
- **Thread 시스템**: 모든 작업은 스레드(Thread) 단위로 관리됨.
- **Affix persona**: 접두사/핵심/접미사 조합의 에이전트 페르소나 생성.
- **Event-driven**: 모든 상태 변화는 EventBus를 통해 전파.

### 3. AXON Studio UI Specification
- **Layout**: 2-column "Sidebar-Main" structure (Left-Aligned).
- **Sidebar (350px)**: 
    - **Office (HR Management)**: Real-time agent roster with **Hire (+) / Fire (-)** controls for each tier.
    - **Colosseum**: Thread List for monitoring and intervening in agent battles.
- **Main Area**: Expanded display for Nogari Lounge (Event/Battle Log) and Bug Repository.
- **Persona Display**: Full identity including generated prefixes and suffixes (e.g., `[Prefix] [Name] [Suffix]`).
