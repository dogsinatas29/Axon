# SYNAPSE v0.0.24 Execution Validator

## Goal

Ensure generated code is both compilable and executable.

------------------------------------------------------------------------

## Pipeline Position

LLM → Pre-clean → Validator → Execution Validator → Write

------------------------------------------------------------------------

## 1. Compile Check

### Rust

-   cargo check

### Python

``` python
import subprocess

def compile_check(path):
    result = subprocess.run(
        ["cargo", "check"],
        cwd=path,
        capture_output=True,
        text=True
    )
    return result.returncode == 0, result.stderr
```

------------------------------------------------------------------------

## 2. Run Check

``` python
def run_check(path):
    result = subprocess.run(
        ["python", "main.py"],
        cwd=path,
        capture_output=True,
        text=True,
        timeout=5
    )
    return result.returncode == 0, result.stdout, result.stderr
```

------------------------------------------------------------------------

## 3. Integration

``` python
def execution_validate(path):
    ok, err = compile_check(path)
    if not ok:
        return False, err

    ok, out, err = run_check(path)
    if not ok:
        return False, err

    return True, "OK"
```

------------------------------------------------------------------------

## 4. Success Criteria

-   compile success
-   runtime success
-   no crash
