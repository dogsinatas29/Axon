# SYNAPSE v0.0.24 Markdown Contamination Solution

## Goal

Eliminate markdown contamination completely from LLM-generated source
code.

------------------------------------------------------------------------

## Problem Definition

LLM outputs include markdown syntax such as:

    ```rust
    fn test() {}


    This results in:
    - Syntax errors
    - Compilation failure

    ---

    ## Root Cause

    - LLM trained on markdown-heavy datasets
    - Default formatting bias toward fenced code blocks

    ---

    ## Solution Strategy (3-Layer Defense)

    ### 1. Validator Enforcement (Mandatory)

    Reject any content containing markdown fences.

    ```python
    def validate(content: str):
        if "```" in content:
            return False
        return True

------------------------------------------------------------------------

### 2. Pre-clean Stage (Recommended)

Strip markdown before validation.

``` python
def strip_markdown(content: str):
    return "\n".join(
        line for line in content.splitlines()
        if not line.strip().startswith("```")
    )
```

Pipeline:

generate → clean → validate → write

------------------------------------------------------------------------

### 3. Output Constraint (Supplemental)

Prompt-level rule:

-   Output MUST be raw source code
-   Markdown syntax is forbidden

(Note: Not sufficient alone, validator is required)

------------------------------------------------------------------------

## Test Harness

### Test 1: Markdown Rejection

``` python
def test_markdown_rejection():
    code = """```rust
fn test() {}
```"""
    assert validate(code) == False
```

------------------------------------------------------------------------

### Test 2: Clean Code Pass

``` python
def test_clean_code():
    code = "fn test() { println!(\"ok\"); }"
    assert validate(code) == True
```

------------------------------------------------------------------------

## Success Criteria

-   Markdown contamination = 0%
-   Validator rejects 100% of contaminated outputs
-   Clean code passes 100%

------------------------------------------------------------------------

## Summary

Markdown contamination is a formatting-layer defect.

Solution:

-   Detect → Reject
-   Optional Clean → Reduce retries

This is the simplest and fastest stabilization step.
