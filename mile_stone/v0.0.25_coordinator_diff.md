# 🧠 AXON v0.0.25 Coordinator Integration (DIFF)

## 🎯 Goal
Replace worker-driven execution with Coordinator-controlled scheduling.

---

## 1. Add coordinator.rs

```rust
use std::collections::{HashMap, HashSet};

#[derive(Clone)]
pub struct Task {
    pub id: String,
    pub target_file: String,
    pub dependencies: Vec<String>,
    pub status: TaskStatus,
}

#[derive(Clone, PartialEq)]
pub enum TaskStatus {
    Pending,
    Ready,
    Running,
    Completed,
    Failed,
}

pub struct Coordinator {
    pub tasks: Vec<Task>,
    pub completed: HashSet<String>,
    pub file_locks: HashMap<String, bool>,
}

impl Coordinator {
    pub fn new(tasks: Vec<Task>) -> Self {
        Self {
            tasks,
            completed: HashSet::new(),
            file_locks: HashMap::new(),
        }
    }

    fn is_ready(&self, task: &Task) -> bool {
        task.dependencies.iter().all(|d| self.completed.contains(d))
    }

    pub fn next_task(&mut self) -> Option<Task> {
        for task in &mut self.tasks {
            if task.status == TaskStatus::Pending && self.is_ready(task) {
                let locked = self.file_locks.get(&task.target_file).cloned().unwrap_or(false);

                if !locked {
                    self.file_locks.insert(task.target_file.clone(), true);
                    task.status = TaskStatus::Running;
                    return Some(task.clone());
                }
            }
        }
        None
    }

    pub fn complete(&mut self, task_id: &str, success: bool) {
        if let Some(task) = self.tasks.iter_mut().find(|t| t.id == task_id) {
            self.file_locks.insert(task.target_file.clone(), false);

            if success {
                task.status = TaskStatus::Completed;
                self.completed.insert(task.id.clone());
            } else {
                task.status = TaskStatus::Failed;
            }
        }
    }
}
```

## 2. Modify daemon execution loop

### BEFORE
```rust
let task = self.pop_ready_task();
self.execute(task);
```

### AFTER
```rust
let mut coordinator = Coordinator::new(self.load_tasks_from_db());

loop {
    if let Some(task) = coordinator.next_task() {
        let result = self.execute(task.clone());
        coordinator.complete(&task.id, result.is_ok());
    } else {
        break;
    }
}
```

## 3. Modify worker loop

### BEFORE
```rust
fn worker_loop() {
    let task = db.pop_ready_task();
    execute(task);
}
```

### AFTER
```rust
fn worker_loop(coordinator: Arc<Mutex<Coordinator>>) {
    loop {
        let task = {
            let mut c = coordinator.lock().unwrap();
            c.next_task()
        };

        match task {
            Some(t) => {
                let result = execute(t.clone());

                let mut c = coordinator.lock().unwrap();
                c.complete(&t.id, result.is_ok());
            }
            None => break,
        }
    }
}
```

## 4. Critical Rules

- target_file MUST exist
- One file = one worker
- No dependency skip
- Coordinator is SSOT for execution
