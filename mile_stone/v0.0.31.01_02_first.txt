# AXON v0.0.31.01 — Required Work Breakdown

현재 필요한 것은 “validator 추가”가 아니다.

진짜 작업은:

```text id="i1c4vp"
semantic authority elevation
```

이다.

즉 semantic validation을:

```text id="d8q2km"
late-stage validator
```

에서:

```text id="n4v7rs"
pipeline root authority
```

로 승격시키는 작업이다.

---

# 1. 가장 먼저 해야 하는 작업

## Spec Semantic Layer 분리

새 구조:

```text id="z3p8mx"
crates/axon-ir/src/spec_semantic/
 ├── mod.rs
 ├── validator.rs
 ├── ontology.rs
 ├── rust.rs
 ├── c.rs
 └── python.rs
```

핵심:

```text id="u5r9wy"
IR validator와 완전히 분리
```

해야 한다.

절대:

```text id="m2x6cb"
validator/mod.rs 안에 spec validator 추가
```

같이 가면 안 된다.

그건 다시 generic validator contamination이 된다.

여기서 중요한 건:
