# STEP 2 — SemanticOntology 구축

먼저 ontology부터 만든다.

이유:

```text id="f8r2tw"
validator보다 ontology가 semantic authority source
```

이기 때문이다.

예:

```rust id="d4m7vx"
pub struct SemanticOntology {
    pub forbidden_terms: &'static [&'static str],
    pub required_terms: &'static [&'static str],
}
```

초기에는 단순해야 한다.

---

# STEP 3 — RustOntology 작성

가장 먼저 Rust부터 만든다.

예:

```rust id="m1q9zc"

    ".h",
    ".hpp",
    "CMakeLists.txt",

    "translation unit",
```


핵심:


이다.
