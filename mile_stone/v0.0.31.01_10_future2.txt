# v0.0.31.01 — Semantic Governance Layer 구현 준비 정리

현재 AXON은:

```text id="m4r8tx"
language-aware runtime
```

까지는 도달했다.

하지만 아직:

```text id="u7m2pk"
semantic authority
```

가 IR 이후에 존재한다.

v0.0.31.01의 목표는 이것을:

```text id="x1v6qs"
IR 이전(pre-IR)
```

으로 끌어올리는 것이다.

---

# 1. 현재 미구현 핵심 계층

남은 핵심 작업:

| 영역                                | 목적                            |
| --------------------------------- | ----------------------------- |
| Spec Semantic Authority           | spec 자체 semantic legality 검증  |
| Pre-IR Semantic Rejection         | invalid semantic intent 조기 차단 |
| Ontology-driven Validation        | validator를 ontology 기반으로 승격   |
| Semantic Contamination Preemption | contamination 자체를 발생 이전에 제거   |

---

## SemanticOntology trait

예상:

```rust id="q1m5rs"
pub trait SemanticOntology {
    fn language(&self) -> Language;

    fn forbidden_terms(&self) -> &[&str];

    fn allowed_task_kinds(&self) -> &[LanguageTaskKind];

}
```

# 30. 최종 정리

지금부터 필요한 작업의 본질은:

```text id="p5r8tx"
validator 추가
```

가 아니다.

진짜 목표는:

```text id="u9m2pk"
semantic authority를 IR 이전으로 이동
```

시키는 것이다.

즉 AXON을:

```text id="v4r7qs"
post-validation runtime
```

에서:

```text id="n6m1tx"
preemptive semantic operating system
```

으로 승격시키는 작업이다.
