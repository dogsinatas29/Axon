## 🧠 상세 설계 및 로직
- **Prompt Hardening**: Architect 에이전트가 예시(main/utils)에 매몰되지 않고 Spec 전체를 파싱하도록 프롬프트 구조 개편.
- **Post Traceability**: Junior의 제안과 Senior의 리뷰를 DB에 영구 저장하여 Studio UI 게시판에서 실시간 확인 가능.
- **Physical Sync**: 빌드 단계 진입 전 모든 Artifact의 물리적 존재 여부를 검증하고, 재작업(Rework) 루프를 가시화.

## 🛠️ 기술적 변경 사항
- **Agent Update:** `generate_ir_with_context` 내의 SKELETON 프롬프트 강화.
- **Daemon Update:** `handle_assignment` 내의 Post 저장 및 파일 쓰기 로직 복구/강화.
- **Storage Update:** Active Task 카운트 시 `Failed` 상태 포함 (재작업 중 빌드 조기 시작 방지).
