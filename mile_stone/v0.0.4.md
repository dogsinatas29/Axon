
# AXON Specification
## v0.0.4

### Safety Guard System

AXON introduces a three‑layer safety system.

Rule 1
Senior approval is required before any file modification can be accepted.

Rule 2
Even with Senior approval, if the Architect detects that a change may break the architecture,
the system pauses and waits for explicit user approval.

Rule 3
If review delays occur, the system guarantees structural stability of files until the user gives final approval.

### Core Principle

User code review is mandatory.

AI agents cannot finalize structural code changes without the user's review.
