
# AXON Specification
## v0.0.5

### Runtime Model

AXON runs as a daemon.

Command:

axon run

Components:

Scheduler
Dispatcher
Agent Runtime
EventBus
Storage

### Project Execution Model

Free tier:

1 active project at a time.

Users may create unlimited projects but cannot run multiple simultaneously.

Paid tier allows multiple concurrent projects.

### Workspace Layout

project-root
 ├ src
 ├ tests
 ├ docs
 └ axon

axon/
 ├ threads
 ├ sandbox
 ├ patches
 └ logs

### Sandbox Model

AI agents never modify the real project source.

All work happens inside sandbox environments.

sandbox/thread-id/

### Diff Generation

After modifications AXON compares:

project source
vs
sandbox source

and generates a diff.

### Review Pipeline

Junior
↓
Senior
↓
Architect
↓
User

### Approve & Lock

User approval finalizes changes and updates architecture.md.

### Development Philosophy

AI writes code.
Human owns code.
