# 🚀 AXON Release Notes - v0.0.21

## [v0.0.21] - 2026-04-29

### 🛡️ Factory Hardening (Self-Correction)
- **Zero-Spec Protection**: Implemented a mandatory validation check in `AgentRuntime` to ensure `architecture.md` contains the required `AXON:SPEC` or `spec` blocks. Agents will now automatically retry up to 5 times if these blocks are missing.
- **Looping/Repetition Guard**: Added a heuristic check for small model "explosions" (e.g., repeating '#' or 'sharp' characters). If detected, the generation is rejected and retried with a corrective hint.
- **Robust IR Parsing**: Relaxed the `axon_ir_compiler.py` regex to allow optional spaces after spec types, making it more resilient to LLM formatting variability.

### 🧠 Prompt Optimization
- **English-First Structure**: Updated the small model prompt to use English for structural markers while preserving the user's preferred locale for content, reducing cognitive load on 1.8B-class models.
- **Direct Directive**: Added "CRITICAL" markers to mandatory mapping blocks in the architect prompt.

### 🛠️ Technical Fixes
- **PathBuf Formatting**: Verified and ensured all `PathBuf` usages in `axon-daemon` use `.display()` for safe logging.
