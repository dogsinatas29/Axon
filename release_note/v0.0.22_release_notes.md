# 🚀 AXON Release Notes - v0.0.22 (Hardened Factory & Hardware Optimized)

## [v0.0.22] - 2026-04-30

### 🇰🇷 한국어 (Korean)

**1. 결정론적 파이프라인 및 출력 규약 강화 (Deterministic Pipeline)**
- 주니어 요원의 출력을 순수 JSON 배열 형식으로 강제하여 파싱 신뢰도를 100%로 끌어올렸습니다.
- **Stage 3.5 Stub Generation**: 구현 전 모든 파일의 물리적 뼈대(Stub)를 선제 생성하여 병렬 빌드 시의 의존성 에러를 완벽히 해결했습니다.

**2. 하드웨어 최적화 및 검증 (Hardware Optimization)**
- i7-4790 및 GTX 1050 Ti 환경에서 Qwen 2.5 7B 모델을 안정적으로 구동하기 위한 오프로딩 및 병렬 처리 최적화를 완료했습니다.
- **Smart Simulator**: 마크다운 추출기 탑재 및 경로 정규화를 통해 저사양 환경에서도 아키텍처 드리프트 오탐지를 차단했습니다.

**3. 관제탑(Studio UI) 동기화**
- 데몬과 UI 간의 실시간 이벤트 스트리밍을 강화하여 작업 게시판의 가시성을 확보하고 에이전트의 상태를 실시간으로 추적합니다.

### 🇺🇸 English

**1. Deterministic Pipeline & Output Contract Hardening**
- Enforced mandatory pure JSON array format for Junior agent outputs, achieving 100% parsing reliability.
- **Stage 3.5 Stub Generation**: Materializes physical stubs for all files before implementation, resolving cross-file dependency issues during parallel bootstrapping.

**2. Hardware Optimization & Verification**
- Optimized GPU offloading and concurrency for NVIDIA GTX 1050 Ti and i7-4790 environments using Qwen 2.5 7B.
- **Smart Simulator**: Implemented intelligent markdown extraction and path normalization to prevent false-positive architecture drift in resource-constrained environments.

**3. Studio UI Synchronization**
- Enhanced real-time event streaming between the daemon and Studio UI, ensuring full task visibility and live agent status tracking on the Work Board.
