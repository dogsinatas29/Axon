# 📦 v0.0.25 Release Notes - The Great Wall

## [요약 / Summary]
AXON 팩토리의 물리적 무결성(Physical Integrity)과 보안을 위한 강력한 보호막(The Great Wall)이 구축되었습니다. LLM 생성 코드의 오염을 방지하고, 작업 범위를 엄격히 제한하며, 검증 프로세스를 최적화하여 생산성을 향상했습니다.

---

## [변경 사항 / Changes]

### 1. Markdown Elimination (마크다운 정제)
- **Problem:** LLM이 응답에 마크다운 백틱(```)을 포함하여 소스 코드가 오염되는 문제.
- **Solution:** `axon-agent` 단계에서 마크다운 정제 로직을 통합하여 순수 소스 코드만 SSOT에 반영되도록 보장.

### 2. Scope Control (Phase 2 - 범위 제어 탐지)
- **Problem:** 요원(Junior)이 권한 없이 새로운 모듈을 생성하거나 다른 파일에 접근하려는 시도.
- **Solution:** `mod`, `use crate::`, `../` 등 금지된 패턴을 실시간으로 탐지하여 위반 시 태스크 즉시 기각.

### 3. Physical Write Gate (물리적 쓰기 게이트)
- **Problem:** 시뮬레이션 결과가 타겟 파일 외의 파일을 수정하려 할 때 발생하는 부수 효과.
- **Solution:** 데몬의 커밋 단계에서 오직 할당된 `target_file`만 물리적 쓰기가 가능하도록 강제 필터링.

### 4. Execution Validator Optimization (검증 최적화)
- **Problem:** 모든 파일 수정 시마다 무거운 실행 하네스를 돌려 발생하는 시간 낭비 및 중간 단계의 허위 실패.
- **Solution:** 프로젝트의 마지막 태스크에서만 전체 시스템 검증을 수행하도록 최적화.

---

## [영향도 / Impact]
- **Stability:** 소스 코드 오염 및 무단 파일 수정 차단으로 시스템 안정성 획기적 향상.
- **Performance:** 불필요한 검증 루프 제거로 전체 공정 속도 약 80% 향상.
- **Security:** 에이전트의 작업 범위를 샌드박스 내 단일 파일로 엄격히 제한.
