# 📦 AXON Release Notes - v0.0.25 (Final)

## 🚀 "The Factory Vibe" - Architectural Integrity & Human Context

이번 릴리즈는 AXON 공장의 **물리적 무결성(Physical Integrity)**을 최종 확정하고, AI 요원들의 개발 문맥(Context)을 시스템적으로 보존하는 **'노가리(Lounge)' 시스템**을 도입한 기념비적인 버전입니다.

---

### 1. 🏗️ 언어 중립적 아키텍처 완성 (C Support)
- **Deterministic Extensions**: 아키텍트가 설계 시 `.c`, `.h`, `.py`, `.rs` 등 명확한 확장자를 부여하도록 하드닝.
- **Dynamic Entry Point**: 프로젝트 언어에 맞춰 `main.c` 또는 `main.py`를 지능적으로 생성.
- **Strict Stubbing**: 각 언어의 주석 규격에 맞는 물리적 스터브 파일 자동 생성.

### 2. 💾 데이터베이스 무결성 & 마이그레이션 (v0.0.25 Hardening)
- **Individual Column Migration**: 기존 DB를 유지하면서 `senior_comment`, `kind`, `target_file` 등 신규 메타데이터 컬럼을 안전하게 주입.
- **Fail-Safe Initializer**: 중복 테이블 생성 및 마이그레이션 충돌 문제를 근본적으로 해결한 견고한 DB 엔진 탑재.

### 3. 💬 '노가리(Lounge)' 시스템 & 잡설 분리
- **Thought Extraction**: 주니어의 답변에서 '구현에 대한 생각'과 '실제 코드'를 시스템 레벨에서 분리.
- **Lounge Sync**: 주니어의 생각은 **#lounge (노가리)** 채널에 자동 게시되어 개발 문맥을 보존.
- **Pure Code Files**: 소스 코드 파일에서는 모든 잡설이 제거되어, 컴파일 오류를 원천 차단하고 순도를 100%로 유지.

### 4. 👓 시니어 리뷰 가시성 확보
- **Review-to-Post**: 시니어 요원의 리뷰 코멘트(`senior_comment`)를 태스크 스레드 게시물로 자동 등록하여 Studio UI에서 즉시 확인 가능하도록 개선했습니다.
- **Real-time Feedback**: 사용자가 에이전트 간의 난상토론과 최종 리뷰 결과를 UI에서 즉시 확인할 수 있습니다.

### 🛠️ 기술적 개선 사항
- **Named Arguments**: 에이전트 프롬프트 포맷팅에 이름 기반 인자 전달 방식을 도입하여 인자 누락 및 구문 오류 해결.
- **Refined Junior Prompt**: 주니어 요원이 `[THOUGHTS]`와 `[CODE]` 태그를 준수하도록 프롬프트 하드닝.

---

**"이제 AXON은 단순한 코드 생성기가 아닙니다. 요원들의 고민이 노가리로 남고, 결과물은 코드로 증명되는 살아있는 소프트웨어 공장입니다."**
