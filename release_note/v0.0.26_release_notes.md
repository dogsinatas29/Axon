# 🚀 Axon Release Notes - v0.0.26 (Hardened & Resilient)

## 📦 Version: v0.0.26
## 📅 Date: 2026-05-06

---

### 🌟 Key Highlights

#### 1. Deterministic State Machine Pipeline
부트스트랩 파이프라인을 7단계의 명확한 상태 머신으로 개편하여 공정의 결정론적 제어를 실현하였습니다.

#### 2. Self-Healing Task Decomposition & I/O Hardening
**[NEW]** LLM의 비정형 출력(Markdown, Extra text)을 정제하는 `JSON Extractor`와 빈 응답을 차단하는 `Empty Guard`를 도입하여 파이프라인의 물리적 안정성을 극대화하였습니다.

#### 3. Intelligence Feedback Loop (Hint Injection)
**[NEW]** 실패 원인을 분석한 `Hint`를 다음 시도에 즉시 주입하여, 에이전트가 실수를 실시간으로 교정하고 성공적인 결과물(JSON)을 도출하도록 학습 능력을 부여하였습니다.

#### 4. Precision Diagnostic Engine
`serde_json`이 내뱉는 다양한 에러 메시지를 정밀하게 분석하여 `JsonError`로 분류하고, 최적화된 처방전을 발행합니다.

---

### 🛠️ Technical Changes

- **Agent Hardening**: `AgentRuntime`에 `extract_json` 필터 이식 및 힌트 주입 프롬프트 엔진 탑재.
- **Daemon Resilience**: `run_v3` 상태 머신에 폐쇄형 피드백 루프(Closed-loop Feedback) 적용.
- **Syntax Stabilization**: 멀티라인 프롬프트 및 이스케이프 구문 오류 해결.

---

### 🐞 Bug Fixes

- **Infinite Loop Elimination**: `Skeleton` 단계에서 발생하는 JSON 파싱 에러로 인한 무한 루프 문제를 정제 필터와 힌트 주입으로 해결.
- **Log Transparency**: 실패 시 **RAW LLM OUTPUT** 전체를 로그에 남겨 디버깅 가시성 확보.
