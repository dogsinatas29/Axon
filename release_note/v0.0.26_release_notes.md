# 🚀 Axon Release Notes - v0.0.26 (The Iron Pipeline)

## 📦 Version: v0.0.26
## 📅 Date: 2026-05-06

---

### 🌟 Key Highlights

#### 1. Deterministic State Machine Pipeline
부트스트랩 파이프라인을 7단계의 명확한 상태 머신으로 개편하여 공정의 결정론적 제어를 실현하였습니다.

#### 2. 3-Layer JSON Stabilization (The Iron Pipeline)
**[NEW]** 출력 채널의 비결정성을 극복하기 위해 3중 방어막을 구축하였습니다:
- **Layer 1 (Sampling)**: 전 드라이버 `temperature: 0.0` 및 `stop` 토큰을 강제하여 무작위성 제거.
- **Layer 2 (Envelope)**: `<JSON_START>` / `<JSON_END>` 토큰을 통한 물리적 출력 경계 확정.
- **Layer 3 (Repair)**: 1차 파싱 실패 시 즉각적인 자가 수복(`repair_ir_pass`)을 통해 파이프라인 무중단 유지.

#### 3. Intelligence Feedback Loop (Hint Injection)
실패 원인을 분석한 `Hint`를 다음 공정에 즉시 주입하여, 에이전트가 실시간으로 교정된 도면을 도출하도록 학습 능력을 부여하였습니다.

---

### 🛠️ Technical Changes

- **Model Hardening**: Gemini, Claude, OpenAI, Ollama 드라이버의 결정론적 파라미터 튜닝 완료.
- **Agent Resilience**: 2-Phase 파싱 전략 및 Hard Envelope 추출기 탑재.
- **Self-Healing**: 파싱 에러 발생 시 자동으로 수리 전문 요원을 호출하는 자가 수복 시스템 구축.

---

### 🐞 Bug Fixes

- **Non-deterministic Failure**: 모델의 잡설이나 마크다운 코드 블록으로 인한 파이프라인 붕괴 현상을 원천 차단.
- **Log Transparency**: 실패 시 **RAW LLM OUTPUT** 전체를 로그에 남겨 디버깅 가시성 확보.
