# 🚀 Axon Release Notes - v0.0.26 (Hardened)

## 📦 Version: v0.0.26
## 📅 Date: 2026-05-06

---

### 🌟 Key Highlights

#### 1. Deterministic State Machine Pipeline
부트스트랩 파이프라인을 7단계의 명확한 상태 머신으로 개편하여 공정의 결정론적 제어를 실현하였습니다.

#### 2. Self-Healing Task Decomposition
`HeaderGen` 및 `ImplGen` 단계에서 발생하는 LLM의 비정형 출력(Malformed JSON)을 즉시 종료하지 않고 진단 후 자동 복구하는 **'에러 흡수 루프'**를 적용하였습니다.

#### 3. Precision Diagnostic Engine
`serde_json`이 내뱉는 `expected value`, `eof while parsing`, `trailing characters` 등의 에러 메시지를 정밀하게 분석하여 `JsonError`로 분류하고, 최적화된 수정 힌트를 모델에게 전달합니다.

#### 4. Physical Build & Runtime Executor
실제 CMake 빌드 및 바이너리 실행 결과를 파이프라인의 피드백으로 사용하여 이론이 아닌 '동작' 중심의 검증 시스템을 구축하였습니다.

---

### 🛠️ Technical Changes

- **Trait Impl**: `Stage`, `RetryScope`, `FixStrategy` 등에 `Copy` 트레이트를 추가하여 소유권 충돌 방지 및 상태 전환 효율화.
- **Log Hardening**: 파싱 실패 시 실제 에러 메시지를 로그에 노출(`error="..."`)하여 가시성 확보.
- **Escalation Ladder**: 패치 실패 시 단계적으로 대응 수위를 높이는 계층적 복구 전략 이식.

---

### 🐞 Bug Fixes

- **Fatal Error Bypass**: `?` 연산자로 인한 파이프라인 조기 종료 문제 해결.
- **Diagnostic Triage**: 원인을 알 수 없던(`Unknown`) 파싱 에러를 정확한 `JsonError`로 분류 성공.
