# 📝 Release Notes v0.0.28 - Stabilization & Syntax Hardening

---

## 🛠️ 주요 변경 사항

### 1. [Fix] axon-daemon 구문 오류 해결 (Syntax Error Fix)
- `crates/axon-daemon/src/lib.rs`의 `ImplGen` 단계에서 중복된 `Err(e)` 블록 및 잘못된 중괄호 배치를 수정하였습니다.
- 이로 인해 발생하던 `unexpected closing delimiter` 컴파일 에러를 해결하였습니다.

### 2. [Fix] Type Mismatch 및 API 불일치 해결
- `verify_with_senior` 호출 시 `tokio::time::timeout`의 `Elapsed` 에러와 `anyhow::Error`가 동일한 매치 암(arm)에서 바인딩되어 발생하던 타입 불일치 문제를 해결하였습니다.
- `ModelDriver` 트레이트에 `id()` 메서드를 추가하여 시니어 요원의 모델명을 로깅에 사용할 수 있도록 개선하였습니다.

### 3. [Clean] 불필요한 임포트 제거
- `crates/axon-agent/src/lib.rs`에서 사용되지 않는 `HashMap`, `HashSet` 임포트를 제거하였습니다.

### 4. [Milestone] v0.0.28 마일스톤 동기화
- 사용자 요청에 따라 마일스톤 버전을 `0.0.28`로 지정하고, `Control & Isolation` 관련 설계를 마일스톤 문서에 반영하였습니다.
