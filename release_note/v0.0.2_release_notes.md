# Release Notes - v0.0.2 (Modular Framework Transformation)

## Summary
Transformation of AXON from a prototype into a modular "Agent Orchestration Framework". This release establishes a robustfoundation for scalable agent management, persistent task tracking, and a premium real-time monitoring interface.

## Key Changes
- **Modular Workspace Architecture**: Restructured the codebase into a Rust workspace with 6 specialized crates:
  - `axon-core`: Common types, protocols, and shared Event Bus.
  - `axon-daemon`: Long-running orchestrator and CLI entry point.
  - `axon-storage`: Persistent task and event tracking via SQLite.
  - `axon-dispatcher`: Priority-based task queue and worker assignment.
  - `axon-agent`: Advanced Agent Runtime featuring an **Affix System** for randomized personas.
  - `axon-model`: Abstract LLM driver layer with Mock and Reqwest support.
- **AXON Studio (v0.1 UI)**: Replaced the static web viewer with a modern React + Vite dashboard.
  - Real-time updates via WebSockets.
  - **The Colosseum**: Interactive thread grid for monitoring agent activity.
  - **Nogari Lounge**: Agent social simulation and communication hub.
  - **Bug Repository**: Integrated bug tracking panel.
- **Improved Persistence**: Migration from JSON-based state to a robust SQLite database for tasks and event history.
- **CLI Interface**: New command suite: `axon init`, `axon read`, `axon run`, `axon status`.

## Technical Specs
- **Rust Edition**: 2024
- **Persistence**: SQLite (via `rusqlite`)
- **Frontend Stack**: React 18, TypeScript, Vite, Framer Motion, Lucide icons.
- **Communication**: Axum WebSockets + AXP Protocol foundation.
