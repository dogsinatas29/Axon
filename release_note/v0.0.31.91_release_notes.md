# v0.0.31.91 Release Notes - Sandbox Path Resolution Fix

## 기술적 변경 사항
- **샌드박스 루트 경로 결정 로직 개선**:
  - `axon-daemon`이 실행되는 작업 디렉토리(CWD)가 이미 대상 프로젝트 폴더인 상태에서 기동될 때, `project_id`가 중복 결합하여 `TEST4_GTK/TEST4_GTK` 형태의 하위 디렉토리를 이중으로 생성하던 경로 계산 오류를 수정했습니다.
  - 이제 CWD에 의존하지 않고, 인자로 지정된 명세 파일(`spec.md`)의 canonical 물리적 부모 경로(`canonical_spec_path.parent()`)를 100% 결정론적인 단일 진실 공급원(SSOT) 샌드박스 루트로 고정 매핑합니다.
- **클린업 및 자가 복구 완비**:
  - 이중 경로 잔재와 캐시 외의 불필요한 생성 임시 파일들을 리셋하고 깨끗하게 재조립하였습니다.
